## About This Blog

This blog is built with **Blog Starter**, a minimal, open-source Next.js template designed for developers who want a fast, beautiful blog without the bloat.

### Features

- **Static site generation** — blazing fast load times
- **Markdown** — write articles in plain `.md` files
- **Syntax highlighting** — dual light/dark themes via Shiki
- **Mermaid diagrams** — flowcharts and diagrams in your posts
- **Dark mode** — toggle between light and dark themes
- **Accessible** — font size controls for readability
- **SEO ready** — Open Graph, Twitter Cards, sitemap

### Tech Stack

- [Next.js](https://nextjs.org) 16 (App Router, SSG)
- [React](https://react.dev) 19
- [Tailwind CSS](https://tailwindcss.com) v4
- [TypeScript](https://typescriptlang.org)
- [Shiki](https://shiki.style) for syntax highlighting
- [Mermaid](https://mermaid.js.org) for diagrams
